## Phase Goal

**As a** recruiter evaluating the demo, **I want to** upload a document and ask questions grounded in it, **so that** I get cited, trustworthy answers with the retrieval visible and no hallucination when the answer is not in the document.

**This plan's slice:** the ingestion write-path core — the pure functions that turn uploaded bytes into embedded, session-scoped chunks in pgvector. Demoable via `python -m unittest tests.test_rag_ingestion` (mocked embed + pool). The HTTP `/upload` endpoint that calls these functions ships in 03-02.

## Acceptance Criteria

- [ ] `ingest_document(pool, session_id, filename, content, content_type)` extracts text (PDF via pypdf, plain-text via UTF-8), strips invisible chars, chunks, caps at 200, batch-embeds, and inserts one `documents` row + N `document_chunks` rows (RAG-01, RAG-02).
- [ ] `embed_batch` retries with `asyncio.sleep(min(2**attempt, 30))` on HTTP 429 and raises a redacted `RuntimeError` after `EMBED_MAX_RETRIES` (RAG-04).
- [ ] `strip_invisible` removes the zero-width / directional / BOM codepoint ranges from text (RAG-09, ingest side).
- [ ] `pypdf` legitimacy is human-verified before install (supply-chain gate).

    - Constants: `EMBED_URL` = the v1beta `models/gemini-embedding-001:batchEmbedContents` generativelanguage endpoint; `EMBED_BATCH_SIZE = 100`; `EMBED_MAX_RETRIES = 3`; `EMBED_DIM = 768`.
    - `async def embed_batch(texts, api_key)`: build one request object per text with keys `model` = "models/gemini-embedding-001", `content` = parts/text, and `embedContentConfig` = outputDimensionality EMBED_DIM; POST `{"requests": [...]}` with header `x-goog-api-key`; on `response.status_code == 429` await `asyncio.sleep(min(2**attempt, 30))` and continue; else `raise_for_status()` and return `[e["values"] for e in data["embeddings"]]`; assert each vector length == EMBED_DIM. On repeated failure raise `RuntimeError` whose message passes provider text through `redact_secrets`.
    - `async def embed_texts(texts, api_key)`: iterate `range(0, len(texts), EMBED_BATCH_SIZE)`, call embed_batch per slice, extend results.
    - `async def embed_query(text, api_key)`: return `(await embed_texts([text], api_key))[0]`.

RETURNING id, fetch the id; for each `(idx, chunk, vec)` build `vec_str = "[" + ",".join(str(round(x,8)) for x in vec) + "]"` and INSERT into `document_chunks (document_id, session_id, chunk_index, content, embedding, token_count)` VALUES `(%s,%s,%s,%s,%s::vector(768),%s)` with `token_count = len(chunk.split())`. Return `{"status": "truncated" if total > MAX_CHUNKS else ("empty" if not chunks else "ok"), "filename": filename, "chunks_stored": len(chunks), "chunks_skipped": max(0, total - MAX_CHUNKS), "doc_id": str(doc_id)}`. Never open() a file or write bytes to disk.
